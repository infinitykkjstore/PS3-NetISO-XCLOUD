# PS3-NetISO-XCLOUD
This project is just a proof of concept that I made because I wanted to test how the console would react reading all ISOs coming from an external API and its performance.

Unfortunately, even though it works perfectly, the read speeds are very low and it doesn't sustain running the games for very long without freezing. It doesn't deliver a playable experience on the server I used for testing.

I believe that if you have a really good dedicated link, with a good CDN, a faster and more consistent origin than Google Drive, in perfect network scenarios on both the server and client sides, this idea might still be possible to become satisfactory enough to make all games playable without download for the community, but it's just a dream for now.
